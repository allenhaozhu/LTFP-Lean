## Statement

Building block proved honestly: the ramp `relu(x-t) - relu(x-t-1)` is a difference of two ReLUs that lies in [0,1], equals 0 for x ≤ t, equals 1 for x ≥ t+1, and is linear (= x-t) in between (lemmas: `ramp_nonneg`, `ramp_le_one`, `ramp_in_unit_interval`, `ramp_eq_zero_of_le`, `ramp_eq_one_of_ge`, `ramp_eq_linear_of_mem`). Linear combinations of such ramps form a dense subalgebra of C([a,b]). The full density-in-C([0,1]ⁿ) result via Stone-Weierstrass on the algebra generated by ridge functions plus Hahn-Banach separation (`Mathlib.Topology.ContinuousFunction.StoneWeierstrass`, `Mathlib.Analysis.NormedSpace.HahnBanach.Separation`) is a documented Mathlib glue gap — promote when upstream lands.

# Bach textbook excerpt — Universal approximation theorem (Cybenko/Hornik)

**Concept ID:** `universal-approximation`
**Chapter:** Ch 9
**Section:** 9.3, 9.3.1, 9.3.4
**Pages:** 249, 256-265 (book pages; PDF pages 265, 272-281)
**Source:** Bach (2024), *Learning Theory from First Principles*

## Statement

From section 9.3 (p. 256):

> **Universality**: Can we approximate any prediction function with a sufficiently large number of neurons?

From section 9.3.1, *Universal approximation properties* (p. 257):

> Now that we can represent precisely all continuous piecewise affine functions on
> [−R, R], we can use classical approximation theorems for functions on [−R, R]. They
> come in different flavors depending on the norm used to characterize the approximation.
> For example, continuous functions can be approximated by piecewise affine functions
> with arbitrary precision in the L∞-norm (defined as the maximal value of |f(x)| for
> x ∈ [−R, R]) by simply taking the piecewise interpolant from a grid (see quantitative
> arguments in section 9.3.3). With a weaker criterion such as the L2-norm (with respect
> to the Lebesgue measure), we can approximate any function in L2 (see, e.g., Rudin,
> 1987). This can be extended to any dimension d by using the Fourier transform
> representation as
>
>     f(x) = (1/(2π)^d) ∫_{R^d} f̂(ω) e^{iω^⊤ x} dω
>
> and approximating the one-dimensional functions sine and cosine as linear superpositions
> of ReLUs. See a more formal quantitative argument in section 9.3.4.

Bach notes (p. 256, parenthetical):

> universal approximation results exist as soon as the activation function is not a
> polynomial (Leshno et al., 1993).

## Proof (verbatim)

**Step 1 — Exact representation of continuous piecewise affine (CPA) functions
(section 9.3.1, p. 256-257).**

> Since each individual function x ↦ η_j(w_j x + b_j)_+ is continuous piecewise affine, the
> output of a neural network has to be continuous piecewise affine as well. It turns out
> that all continuous piecewise affine functions with m − 2 kinks in the open interval
> (−R, R) can be represented by m neurons on [−R, R].
>
> Indeed, [...] if we assume that the function f is such that f(−R) = 0, with kinks
> a_1 < ··· < a_{m−2} on (−R, R), we can approximate it on [−R, a_1] by the function
> v_1(x + R)_+ where v_1 is the slope of f on [−R, a_1]. The approximation is tight on
> [−R, a_1]. To have a tight approximation on [a_1, a_2] without perturbing the
> approximation on [−R, a_1], we can add to the approximation v_2(x − a_1)_+, where v_2 is
> exactly what is needed to compensate for the change in slope of f. By pursuing this
> reasoning, we can represent the function on [−R, R] exactly with m − 1 neurons.
>
> To remove the constraint that f(−R) = 0, we can simply notice that
> (1/(2R))(x + R)_+ + (1/(2R))(−x + R)_+ is equal to 1 on [−R, R]. Thus, with one
> additional neuron (only one since (x + R)_+ has already been used), we can represent any
> piecewise-affine function with m − 2 kinks using m neurons.

**Step 2 — CPA functions are dense in C([−R, R]) in L∞, and in L²(R) in L².**
Standard classical fact (Bach cites Rudin, 1987). Take the piecewise-linear interpolant
on an ε-grid.

**Step 3 — Extension to arbitrary dimension via Fourier inversion (section 9.3.4,
p. 263-265).**

> Since f ∈ L¹(R^d), the Fourier transform f̂(ω) = ∫_{R^d} f(x) e^{−iω^⊤ x} dx is
> defined everywhere and continuous, and, assuming that Fourier transform is integrable,
> we can write f as the inverse Fourier transform of f̂:
>
>     f(x) = (1/(2π)^d) ∫_{R^d} f̂(ω) e^{iω^⊤ x} dω.       (9.12)
>
> To compute an upper bound on γ_1(f), it suffices to upper-bound for each ω ∈ R^d the
> norm γ_1(x ↦ e^{iω^⊤ x}). [...] we obtain two measures on [−R, R], ν_+^(ρ) and ν_−^(ρ),
> such that for all u ∈ [−R, R],
>
>     e^{iuρ} = ∫_{−R}^R (u − b)_+ dν_+^(ρ)(b) + ∫_{−R}^R (−u − b)_+ dν_−^(ρ)(b),
>
> with [total-variation bound] ≤ (2/R)(1 + 2R²ρ²).        (9.13)

Quantitative variation-norm bound (equation (9.14)):

>     γ_1(f) ≤ (1/(2π)^d) (2/R) ∫_{R^d} |f̂(ω)| (1 + 2R²‖ω‖²₂) dω.

The right side is the **Barron norm** (Barron 1993, 1994). Finiteness of the Barron
norm implies finiteness of γ_1 (i.e., representability with bounded ℓ¹-norm of output
weights), hence approximability by finite-width ReLU networks.

## Notes

- **Intermediate lemmas inside the proof:**
  - *CPA exactness:* any CPA function with k kinks on (−R, R) is expressible as a sum
    of k + 2 ReLU neurons. Uses positive homogeneity of ReLU and the partition-of-unity
    identity `(1/(2R))(x+R)_+ + (1/(2R))(−x+R)_+ = 1` on [−R, R].
  - *Single-variable complex exponentials* are γ_1-bounded: γ_1(u ↦ e^{iuρ}) ≤
    (2/R)(1 + 2R²ρ²). Proved via Taylor remainder + equation (9.11).
  - *Fourier inversion* lifts the 1-D bound to d dimensions: γ_1(x ↦ e^{iω^⊤ x}) ≤
    (2/R)(1 + 2R²‖ω‖²₂).
  - *Triangle inequality on γ_1* (equation (9.14)) wraps the per-frequency bound into a
    global Barron-norm bound.
- **Technique in one line:** approximate continuous functions by CPA functions (1-D);
  express CPA functions exactly via ReLU neurons; lift to R^d by Fourier inversion +
  triangle inequality on the variation norm.
- **Strength of the result.** Bach's treatment goes beyond classical (qualitative)
  Cybenko/Hornik: it provides a *quantitative* finite-width bound via the variation /
  Barron norm — the number of neurons needed for L²(p)-error ε is O(γ_1(g)²/ε²)
  (equation (9.18), Frank-Wolfe / Maurey-Jones-Barron / Carathéodory style; section
  9.3.6). Sharpening to ε^{−2d/(d+3)} is referenced (Bach 2017).
- **Hypotheses required.** Activation must be **non-polynomial** (Leshno et al. 1993);
  ReLU satisfies this. For the quantitative Fourier-based bound, f must have Fourier
  transform in L¹(R^d) with the weighted integral ∫|f̂|(1 + 2R²‖ω‖²) dω finite.
- **Ambiguities for Lean formalization.**
  - The textbook does NOT prove the classical Cybenko/Hornik existence statement in
    the abstract general form ("for every continuous f and every ε, there exist m, η, w, b
    such that sup‖f − f̂_m‖ ≤ ε"). It instead proves the *quantitative* variation-norm
    version for Barron-norm-finite functions, plus the *exact* CPA representation.
    A Lean formalization could pick either:
      1. The exact CPA-representation theorem (a constructive 1-D statement;
         immediately formalizable from sections 9.2 + 9.3.1).
      2. The quantitative Fourier-based bound (requires Fourier transform infrastructure
         in Mathlib; significantly heavier).
      3. The Leshno/Cybenko abstract density statement (cited but not proved in the book).
  - Bach's classification of UAT as ♦ (diamond) in the concept registry suggests
    formalization is non-trivial; the natural Lean target is probably the 1-D CPA
    representation since it is the only piece Bach proves in detail and it carries the
    constructive content.
- **Wiki MVP flag.** This concept was identified as a high-stakes *orphan* in the
  concept graph. The book treatment makes clear it should depend on (and consume):
  `relu-positive-homog`, `single-hidden-relu`, `relu-add-le`, the partition-of-unity
  identity `(1/(2R))(x+R)_+ + (1/(2R))(−x+R)_+ = 1` on [−R, R], and (in higher dim)
  Fourier inversion. None of these except the ReLU anchors currently exist as
  prerequisites in the DAG. Adding them — at minimum the CPA-exactness lemma — would
  un-orphan UAT.
